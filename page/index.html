Algorithm Reference
FIFO
First-In First-Out
The oldest page in memory is always evicted first. Simple queue structure. Can suffer from Bélády's anomaly — more frames can mean more faults.
O(n) · Stack O(f)
LRU
Least Recently Used
Evicts the page not used for the longest time. Good approximation of Optimal. Does not suffer from Bélády's anomaly.
O(n) · Stack O(f)
OPTIMAL
Optimal (OPT)
Evicts the page not needed for the longest future time. Requires future knowledge — used only as a theoretical benchmark.
O(n·f) · Theoretical
MFU
Most Frequently Used
Evicts the page with the highest access count, reasoning it has served its purpose. Counter-intuitive — useful for specific workloads.
O(n) · Stack O(f)
LFU
Least Frequently Used
Evicts the page with the lowest access count. Ties broken by LRU order. Works well with stable, predictable access patterns.
O(n) · Stack O(f)
