SYSTEM OPERATIONS

01

Driver Registration: OS maps device memory into the kernel space. Without this, hardware is "invisible" to the CPU.

02

Interrupt Trigger (IRQ): Hardware sends an electrical signal. The CPU pauses current tasks and adds the request to the I/O Queue.

03

Serial Execution: The Kernel executes the Interrupt Service Routine (ISR). It processes only the head of the queue to ensure data integrity.
